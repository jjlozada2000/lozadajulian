/* ---- HOW I USE AI IN MY PROJECTS AND WORKFLOW ---- */
My discipline with using AI (Claude, for reference) for personal projects are structured as:
  1. I first prompt Claude with my objective and the tools I am most familiar with/have on hand.
  2. I section off the parts of the project from what I do understand and do not understand, and then send another prompt asking Claude to explain the sections I do not understand.
  3. I code as much as I can without Claude, writing comments and code in ways that I have learned from school or other projects.
  4. I ask Claude to write a program skeleton of the parts I do not understand, and to further explain it in a more digestable way.
  5. I only ask and use Claude for written code when I fully understand the functionality of what is being written.

For this project, I am using my experience from coding my Senior Project (another repository called "seatech") and showcasing it here.
